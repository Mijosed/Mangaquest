# Mangaquest

## Étapes d'installation

1. **Installer les dépendances :**
    ```bash
    composer install
    ```

2. **Démarrer le serveur Symfony :**
    ```bash
    symfony serve
    ```
3. **Se connecter :**
    - Les identifiants de connexion se trouvent dans `fixtures/user.yaml`.
      
4. **Charger les animes :**
   ```bash
    php bin/console app:fetch-anime
   ```
